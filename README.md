# NPS ESG Watch — 시민감시 대시보드 (프로토타입)

> 가입자 알권리를 위한 국민연금 기금감시 ESG 리스크 조기탐지 대시보드
> 2026 연금행동 신진연구자 학술제 제출용 정적 프로토타입

## 개요

본 대시보드는 **공적연금강화국민행동(PPA) 신진연구자 학술제** 제출 연구제안서의 산출물 ⑤ 「시민감시 대시보드 프로토타입」으로, 본 연구가 학술제 발표 시점(2026.12)에 라이브 데이터로 갱신할 예정인 도구의 **정적 시연 버전**이다.

## 주요 구성

1. **KPI 4종** — 오늘의 ESG 리스크 신호 / 평가기관 불일치 워치 / 의결권 시즌 카운트다운 / 시민감시 신청 건수
2. **모듈 1.** ESG 리스크 신호 Top 5 (NLP 자동 탐지·KCGS 공고 대비 리드타임)
3. **모듈 2.** 평가기관 등급 불일치 워치 (KCGS·Sustinvest·MSCI letter grade)
4. **모듈 3.** NPS 의결권 패턴 시계열 (2018~2025 · DID 시각화)
5. **모듈 4.** 시민감시 큐 (human-in-the-loop · 가입자 모니터링 신청 인터페이스)
6. **글로벌 비교** — GPFG·CalPERS·ABP·CPPIB·NPS 5축 거버넌스 방사형 차트

## 기술 스택

- HTML / CSS / Vanilla JS
- Chart.js 4.4.1 (CDN)
- 단일 파일 정적 배포 (Vercel·GitHub Pages 호환)
